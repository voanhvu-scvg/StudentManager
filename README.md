Hướng Dẫn Sử Dụng
Dự án Quản lý Sinh viên (Student Manager) Đây là một ứng dụng quản lý sinh viên đơn giản được xây dựng bằng C# và Windows Forms (.NET Framework / .NET Core). Yêu cầu Hệ thống
Hệ điều hành: Windows 10/11
Môi trường phát triển: Visual Studio 2019/2022
SDK: .NET Framework 4.7.2 (hoặc phiên bản .NET mà project của bạn sử dụng).

Bắt đầu
Clone Repository về máy của bạn:
git clone [https://github.com/voanhvu-scvg/StudentManager.git](
https://github.com/voanhvu-scvg/StudentManager.git)
Mở Solution file (StudentManager.sln) bằng Visual Studio.
Nhấn Start (F5) để chạy ứng dụng.
Sử dụng các nút Thêm/Sửa/Xóa để thao tác với dữ liệu sinh viên.
